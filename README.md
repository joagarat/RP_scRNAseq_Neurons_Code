# RP_scRNAseq_Neurons_Code
This repository provides all the code and scripts necessary to reproduce the data analysis and generate the figures for the article, "Single-cell RNA-seq Reveals Distinct Ribosomal Protein Expression Profiles Among Neuronal Subtypes".

Data and Code Availability

The single-cell RNA-seq datasets re-analyzed in this study are publicly available. 
The datasets from Yao et al. (2021) and Jin et al. (2025) were sourced from the Neuroscience Multi-omic (NeMO) Archive under the dataset identifiers dat-jb2f34y and dat-61kfys3, respectively. 
The dataset from Hing et al. (2024) is available in the Gene Expression Omnibus (GEO) database under accession number GSE240975.

For reproducibility, we note that the Smart-seq2 dataset was downloaded in Seurat object format (`Seurat.ss.rda`) from the Allen Institute Brain Map portal:
https://brain-map.org/our-research/cell-types-taxonomies/cell-types-database-rna-seq-data/mouse-whole-cortex-and-hippocampus-smart-seq
